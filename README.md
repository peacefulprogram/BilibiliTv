## 概述

本项目为bilibili的android tv客户端,仅供学习使用,不提供apk
## 已实现功能
- [x] 视频推荐
- [x] 关注用户的动态
- [x] 播放历史
- [x] 直播
- [x] 分p视频

未实现功能
- [] 剧集和番剧
